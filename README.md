
# FoodMix

## Descripción del proyecto

FoodMix es una aplicación móvil desarrollada en Flutter que permite a los usuarios consultar, crear y administrar recetas de comida.

La aplicación cuenta con inicio de sesión, registro de usuarios, búsqueda de recetas, recetas aleatorias, favoritos y gestión de recetas creadas por el usuario.

El objetivo de FoodMix es brindar una forma sencilla y organizada de descubrir, guardar y compartir recetas.

---

## Características principales

- Inicio de sesión y registro de usuario.
- Visualización de recetas.
- Búsqueda de recetas.
- Recetas aleatorias.
- Sistema de favoritos.
- Creación de recetas personalizadas.
- Eliminación de recetas creadas por el usuario.
- Vista detallada de cada receta.

---

## Tecnologías utilizadas

- Flutter
- Dart
- Provider para manejo de estado
- Material Design

---

## Requisitos para instalar

Para ejecutar el proyecto se necesita:

- Flutter SDK instalado.
- Android Studio o Visual Studio Code.
- Emulador Android o dispositivo físico.

---

## Instalación

Clonar el repositorio:

```bash
git clone LINK_DEL_REPOSITORIO
````

Entrar a la carpeta del proyecto:

```bash
cd FoodMix
```

Instalar dependencias:

```bash
flutter pub get
```

Ejecutar la aplicación:

```bash
flutter run
```

---

## Uso de la aplicación

Al iniciar FoodMix el usuario puede crear una cuenta o iniciar sesión.

Después puede:

* Explorar recetas disponibles.
* Buscar recetas por nombre.
* Ver recetas aleatorias.
* Agregar recetas a favoritos.
* Crear sus propias recetas.
* Consultar la preparación e ingredientes.
